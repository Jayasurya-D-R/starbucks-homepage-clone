# Starbucks Homepage Clone

A frontend clone of the Starbucks homepage built using HTML5 and CSS3.

## Features

- Responsive homepage layout
- Navigation bar
- Hero section
- Promotional sections
- Footer
- Flexbox-based layouts
- Hover effects and transitions
- Clean and organized CSS

## Technologies Used

- HTML5
- CSS3
- Flexbox

## What I Learned

- Semantic HTML
- CSS Flexbox
- Layout design
- Image positioning
- Hover effects
- CSS transitions
- Building a real-world website from scratch
